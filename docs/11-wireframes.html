Wireframes — Asset Tracking System

Low-fidelity: grayscale + one olive accent, review layout and flow, not final visuals. Both portals are one SPA with role-gated navigation (doc 06). Signature element: the pipeline rail — a document's journey (scan → parse → extract → validate → done) rendered as a conveyor, reused everywhere a document's state appears. Density: workbench-tight (14px body, 8px grid), because custodians work through lists, not brochures.
